# Expense Tracker

A modern mobile-friendly Expense Tracker built with HTML, CSS, JavaScript, Google Sheets, and Google Apps Script.

## Features

* Income & Expense Tracking
* Loan Management
* Recent Transactions
* Transaction History Search
* Dark Mode
* Mobile Responsive UI
* Google Sheets Database
* Pull To Refresh
* Auto Loan Summary
* Bkash Auto Select for DPS Payment

## Categories

### Income

* Salary
* Business
* Bonus
* Extra Work
* Take Loan
* Loan Received

### Expense

* Personal
* Family
* Home Expense
* DPS Payment
* Give Loan
* Loan Paid

## Tech Stack

* HTML
* CSS
* JavaScript
* Google Apps Script
* Google Sheets

## Setup

1. Create a Google Sheet
2. Add Apps Script
3. Deploy as Web App
4. Copy Web App URL
5. Replace `SCRIPT_URL` in `index.html`
